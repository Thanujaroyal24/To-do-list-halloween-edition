# 🎃 Spooky Tasks Tracker 👻

A fun, fully interactive Halloween-themed task manager built with **pure HTML, CSS, and vanilla JavaScript** — no frameworks, no dependencies!

Keep track of assignments, deadlines, reminders, and personal tasks without letting them haunt you. Features live countdowns, priorities, categories, search, filters, and a "graveyard" for completed tasks ⚰️.

## ✨ Features
- Add, edit, complete, and delete tasks
- Categories (Assignments, Deadlines, Reminders, Personal, Additional Works)
- Priority levels (Low 🍬, Medium 🎃, High 💀)
- Due date & time with real-time countdown timers (turns urgent when <24h left)
- Search and filter tasks
- Spooky animations: twinkling stars, floating pumpkins/ghosts/bats, celebration effects on task completion
- "Graveyard" section for completed tasks
- Fully responsive design

## 🖥️ How to Run Locally
1. Clone or download this repository
2. Open `index.html` in any modern browser
3. Start adding your spooky tasks!

No installation or build steps required — it just works!
JUST COPY THE CODE AND RUN IN NOTEPAD++ OR VS CODE🕸️
